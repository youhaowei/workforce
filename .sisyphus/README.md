# .sisyphus Workspace Notes

This folder stores lightweight planning state for the `workforce` project.

## Layout

- `plans/workforce.md`: current implementation plan (active)
- `notepads/workforce/decisions.md`: accepted architecture/product decisions
- `notepads/workforce/problems.md`: unresolved product/engineering decisions
- `notepads/workforce/issues.md`: active blockers and risks
- `notepads/workforce/learnings.md`: compact implementation learnings
- `boulder.json`: tool runtime metadata (active plan pointer, sessions)

## Source of Truth

Product requirements are defined in:

- `docs/PRD-MVP.md`

This folder should stay concise and operational. Historical detail should live in git history or dedicated docs, not repeated across multiple note files.
